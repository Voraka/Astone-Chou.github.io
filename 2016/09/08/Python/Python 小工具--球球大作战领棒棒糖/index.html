　　既然只是一个 Http 访问而已, 那么每天定时访问就可以自动的把任务做完了. 链接的地址大概是这样的 http://t.cn/Rta1ive, 点进去之后会跳转到 battleofballs 的网站上, 上面有自己的 ID 和昵称. 浏览器点开 F12 查看 Network 选项, 发现了关键的链接 http://cn.battleofballs.com/share?type=1&id=174777161. 看样子就是通过这个链接获取棒棒糖了.
