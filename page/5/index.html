Lua 数据结构--Closure

Closure

　　在 lua 中函数是作为第一类值存在的，即它和表、字符串、数值一样可以被变量引用。函数有三种类型 lua 函数 (LUA_TLCL)、轻量级 C 函数 (LUA_TLCF)、C 函数 (LUA_TCCL)。其中 LUA_TLCL 和 LUA_TCCL 属于 GCUnion–需要被回收的类型，被称为 Closure。当 LUA_TCCL 不包含 upvalue 时，直接用 C 函数指针即可，不必构造 Closure 对象, 也就是 LUA_TLCF。

Lua 数据结构--Table(一)

Table

　　Table 是 lua 中唯一的数据结构。它是一个关联数组，也就是说，这个数组不仅仅以数字做索引，除了 nil 和 NaN 之外的所有 lua 值都可以做索引。同时，它也是异构数组，即数据的值也可以是不同的类型。Table 可被用于表示普通数组、序列、符号表、集合、记录、图、树等等。

TString

Lua 数据结构--TValue

TValue

　　TValue 是实现 Lua 弱数据类型的主要数据结构。所有关于 lua 数据值的操作都会涉及到 TValue。因此 TValue 贯穿了整个 Lua 的实现。

　　在 lua.h 中的宏定义列出了 lua 语法中所涉及的所有基础类型。但实际上 lua 在内部实现上会对这些基础数据类型进行更细粒度的划分，例如：LUA_TSTRING 在 lua 的实现层面会分成 LUA_TSHRSTR 和 LUA_TLNGSTR 。
